# AI Daily Digest

每日 AI 行业动态摘要——覆盖热点新闻、企业级实践、AI Coding、深度报告。

🌐 网站：https://dongbocui-maker.github.io/ai-daily/

## 内容范围

每日整理过去 24 小时全球 AI 行业重要动态，按四个板块组织：

- 🔥 **AI 热点新闻** — 重大融资、产品发布、监管动态
- 🏢 **企业级 AI 实践** — 落地案例、Agent 编排、企业架构
- 💻 **AI Coding 动态** — 编码模型、IDE 集成、开发者工具
- 📊 **深度报告与论文** — 行业研究、学术论文、白皮书

## 技术栈

- **Astro** — 静态站点生成器
- **Tailwind CSS** — 样式
- **TypeScript** — 类型安全
- **GitHub Actions** — 每日自动同步

## 本地开发

```bash
pnpm install
pnpm dev          # 启动开发服务器
pnpm build        # 构建静态站点
pnpm sync         # 从飞书同步最新内容（需配 .env）
```
